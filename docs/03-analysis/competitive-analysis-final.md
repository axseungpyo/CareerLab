```

│ [맞춤법검사] [저장기록] [메모장] [친구첨삭] [자소서전문첨삭] [저장] │
├────────────────────────────────────┬─────────────────────────┤

│                                    │                         │

└────────────────────────────────────┴─────────────────────────┘
```

#### 에디터 도구 상세

| 도구 | 기능 |
|------|------|
| 맞춤법검사 | 실시간 검사, 오류 수 표시, 재검사 버튼 |
| 저장기록 | 자동/수동 저장 히스토리 (버전 관리 유사) |
| 메모장 | 사이드 노트 패드 (아이디어/키워드 임시 저장) |
| 친구첨삭 | 공유 링크 생성 → 친구에게 피드백 요청 |
| AI 마스터자소서 | AI 자소서 초안 생성 (에디터 진입 시 자동 시작!) |
| 전문첨삭 | 유료 전문가 첨삭 서비스 연결 |
| 글자수 카운터 | 프로그레스 바 + 공백 포함/제외 설정 + 기업별 제한 자동 적용 |

#### 기업 연결 / 기출문항 로딩
- 기업 선택 시 직무별 자소서 문항 자동 채움 (시즌별 축적)
- 같은 회사 다른 직무 = 다른 문항 세트 (예: 비씨카드 Biz_일반 2,143명 / DX_인턴 880명)
- 문항별 "N명 작성" 소셜 프루프 실시간 표시
- "문항 오류 신고" 크라우드소싱 품질 관리

- **1:1 현직자 과외** (/1on1-tutor): 300명+ 현직자 (삼성/LG/현대/카카오뱅크 등), 4단계 선발
- 가격 비공개 (맞춤 견적)

### 3.9 직무 교육 (/training)
- 국비지원 교육 과정 통합 검색 (KDT, KDC, 새싹)
- IT/제조/건설/경영/안전 카테고리, 훈련장려금 월 최대 31.6만원

### 3.10 장점

1. **기출문항 DB의 깊이** — 수천 기업 × 직무별 × 시즌별, 자동 문항 채움
2. **AI 자소서 자동 초안** — 에디터 진입 시 즉시 생성 시작 (작성시간 35% 단축)
3. **실시간 소셜 프루프** — "2,143명 작성", "70,483회 조회"
4. **4-탭 통합 공고 뷰** — 자소서/공고/기업/지원자를 한 페이지에
5. **칸반 스케줄러** — 자소서 작성 → 전형 과정 연동 관리
6. **채팅 커뮤니티** — 19,674명, 인사담당자 직접 참여
7. **글자수 관리 UX** — 프로그레스 바 + 공백 설정 + 기업별 자동 제한

### 3.11 단점

1. **이력서 기능 없음** — 신입용 이력서 작성 불가 (경력기술서만)
2. **로그인 벽** — 맞춤법, 지원자 분석, 합격후기 모두 로그인 필수
3. **인라인 AI 부재** — 실시간 문장 추천 없음, 별도 "AI 마스터 자소서"
4. **내보내기 제한** — PDF만 확인, Word/HWP 미확인
5. **채팅 사이드바 과부하** — 에디터 우측 항상 열림, 집중 방해
6. **합격 자소서 전문 미공개** — 합격후기는 간략 경험담 수준

## 7. 종합 결론

### CareerLab의 전략적 포지셔닝

Worxphere 생태계(잡코리아+사람인+자소설닷컴)는 **150만 사용자 + 2,500만 자소서 + 7,800건 합격자소서**라는 거대한 데이터 해자(moat)를 보유. CareerLab이 데이터 규모에서 경쟁하는 것은 비현실적.

**CareerLab만의 해자:**
1. **AI 품질** — Claude Sonnet 무제한 생성/첨삭/면접 (경쟁사: 무료 4회+유료)
2. **프라이버시** — 로컬 앱, 광고 없음, 개인정보 미수집
3. **워크플로우 통합** — 프로필→기업분석→자소서→면접→지원관리 원스톱
4. **기술적 차별화** — pgvector 시맨틱 매칭, Diff 뷰어, YAML 프롬프트 시스템

**v0.9 핵심 전략:**
경쟁사의 UX 장점(글자수 관리, 맞춤법, 메모장, 칸반 연동)을 흡수하면서,
AI 품질/면접/버전관리/프라이버시 등 이미 앞선 영역을 더 강화.
