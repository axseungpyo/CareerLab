# 01. 프로젝트 개요

## 프로젝트명
**CareerLab** — AI 커리어 컨설팅 에이전트

## 목적
내 이력/경력 데이터를 기반으로, 지원 기업에 특화된 자기소개서 생성 및 면접 컨설팅을 제공하는 개인용 AI 에이전트

## 배포 범위
로컬 전용 (localhost 웹 앱), 추후 배포 가능하도록 Supabase 기반 설계

## 핵심 기능 4가지

| # | 기능 | 설명 |
|---|------|------|
| 1 | 이력/경력 관리 | STAR 구조 기반 커리어 데이터 체계적 관리 |
| 2 | 기업별 맞춤 자소서 | 채용공고 분석 → 내 경력 매칭 → 문항별 생성 |
| 3 | 면접 코칭 | 예상질문 생성 + 모의면접 채팅 + 평가 리포트 |
| 4 | 자소서 첨삭 | 구조/내용/표현/전략 4축 분석 + diff 뷰어 |

## 사용 시나리오

```
1. 프로필 등록
   → 경력, 프로젝트, 역량을 입력 (폼/파일/대화형)
   → Supabase에 저장 + pgvector 임베딩

2. 기업 분석
   → 채용공고 붙여넣기 or 검색
   → Claude Desktop 리서치 결과 연동
   → LLM이 요구사항/인재상/키워드 자동 추출

3. 자소서 생성
   → 기업 분석 + 내 프로필 시맨틱 매칭
   → 문항별 맞춤 자소서 초안 생성
   → 톤/글자수/강조 포인트 커스터마이징

4. 첨삭
   → 작성된 자소서 업로드
   → 4축 분석 리포트 + 수정 제안
   → 원본 vs 수정본 비교 뷰

5. 면접 준비
   → 자소서 기반 예상질문 자동 생성
   → 모의면접 채팅 (면접관 AI)
   → 답변별 피드백 + 종합 평가
```

## 프로젝트 타임라인

| Phase | 기간 | 내용 |
|-------|------|------|
| 0 | 2주 | 환경 세팅 + 핵심 인프라 (LLM 라우터, OAuth, 프롬프트 엔진) |
| 1 | 1주 | 프로필 관리 CRUD |
| 2 | 2~3주 | 자소서 생성 파이프라인 |
| 3 | 1주 | 첨삭/피드백 |
| 4 | 2주 | 면접 코칭 |
| 5 | 1~2주 | 대시보드 + 고도화 |
