# 06. 기능 상세 설계

## 6.1 이력/경력 데이터 관리

### 데이터 모델

```
Profile
├── 기본 정보 (이름, 연락처, 학력)
├── 경력 사항[]
│   ├── 회사명, 직급, 기간
│   ├── 핵심 업무 설명
│   ├── 성과/실적 (정량 수치)
│   └── 사용 도구/기술
├── 프로젝트[]
│   ├── 프로젝트명, 기간, 역할
│   ├── STAR 구조 (상황→과제→행동→결과)
│   └── 배운 점
├── 역량 키워드[]
│   ├── 하드 스킬 (도구, 기술)
│   └── 소프트 스킬 (리더십 등)
├── 자기 서사
│   ├── 커리어 전환 계기
│   ├── 핵심 가치관
│   └── 장기 목표
└── 기존 자소서 아카이브[]
```

### 입력 경로 4가지

| 경로 | 방식 | 기술 |
|------|------|------|
| 직접 입력 | 폼 UI | shadcn/ui Form + Supabase insert |
| 파일 업로드 | PDF/DOCX 파싱 | GPT-4o-mini → 구조화 JSON |
| Google Drive | 기존 이력서 연동 | Google Drive API |
| 대화형 입력 | AI 인터뷰 | Claude 채팅으로 경력 정리 |

### 핵심 기능
- STAR 구조 자동 제안 ("이 경력을 STAR로 정리해볼까요?")
- 성과 정량화 유도 ("매출 증가"→ "전년 대비 15% 매출 증가")
- 입력 즉시 벡터 임베딩 → pgvector 저장

## 6.2 기업별 맞춤 자소서 생성

### 워크플로우

```
[1] 기업 정보 입력
    ├── 채용공고 텍스트 붙여넣기
    ├── 기업 홈페이지 URL (선택)
    ├── Claude Desktop 리서치 결과 (선택)
    └── Brave Search 자동 검색
         │
[2] 기업 분석 (GPT-4o-mini)
    ├── 채용공고 핵심 요구사항 추출
    ├── 인재상/핵심가치 파악
    ├── 직무 키워드 매핑
    └── 자소서 문항 분석
         │
[3] 내 데이터 매칭 (pgvector)
    ├── 요구사항 ↔ 내 경력 시맨틱 매칭
    ├── 기업 가치 ↔ 내 가치관 매칭
    └── 활용할 에피소드 후보 추천
         │
[4] 자소서 초안 생성 (Claude Sonnet, 스트리밍)
    ├── 문항별 맞춤 답변
    ├── 기업 톤앤매너 반영
    └── STAR 구조 적용
         │
[5] 검토 & 수정 (UI)
    ├── 글자 수 실시간 카운트
    ├── 키워드 매칭률 표시
    └── 재생성 / 부분 수정
```

### 생성 옵션
- 톤: 진중한 / 열정적 / 전문적 / 친근한
- 강조 경력: 사용자가 직접 선택
- 글자 수: 문항별 제한 설정
- 버전 관리: 같은 기업 여러 버전 비교

## 6.3 면접 예상질문 + 모의면접

### Phase 1: 예상질문 생성

```
입력: 자소서 + 채용공고 + 기업 분석
         │
    질문 카테고리별 생성 (Claude Haiku)
    ├── ① 자소서 기반 ("~라고 쓰셨는데, 구체적으로?")
    ├── ② 직무 역량 ("이 업무에서 KPI 관리 경험?")
    ├── ③ 기업 맞춤 ("우리 회사 최근 사업에 대해?")
    ├── ④ 인성/가치관 ("팀 갈등 해결 경험?")
    └── ⑤ 돌발/압박 ("이직이 잦은 이유는?")
         │
    질문별 답변 가이드 (내 데이터 기반)
```

### Phase 2: 모의면접

- 채팅 인터페이스 (Vercel AI SDK useChat)
- 면접 모드:
  - 일반 면접: 순차적 질문
  - 압박 면접: 꼬리질문, 반박
  - PT 면접: 주제 제시 → 답변 → 질문
- 세션 종료 후 평가:
  - 답변별 점수 (구체성, 논리성, 직무 관련성)
  - 개선 포인트 + 모범답변
  - 답변 길이/시간 분석

## 6.4 자소서 첨삭/피드백

### 분석 프레임워크 (4축)

| 축 | 분석 항목 | 점수 |
|---|----------|------|
| 구조 | 서론/본론/결론 흐름, STAR 충실도, 문항 의도 대응 | 1~10 |
| 내용 | 구체성(수치/사례), 차별화 요소, 기업 맞춤도 | 1~10 |
| 표현 | 문장 간결성, 반복 표현, 진부한 표현 감지 | 1~10 |
| 전략 | 직무 키워드 포함도, 면접 유도 전략 | 1~10 |

### 출력 형태
- 인라인 첨삭: 원문에 수정 제안 마킹
- 종합 리포트: 4축 레이더 차트 + 개선 우선순위
- diff 뷰어: 원본 vs 수정본 나란히 비교

### 진부한 표현 감지 목록 (프롬프트에 포함)
```
- "열정을 가지고" → 구체적 행동으로 대체
- "노력하겠습니다" → 구체적 계획으로 대체
- "도전정신" → 실제 도전 사례로 대체
- "소통을 잘하는" → 소통한 구체적 상황으로 대체
- "4차 산업혁명" → 구체적 기술/트렌드로 대체
```
